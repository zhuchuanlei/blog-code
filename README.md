# blog-code
项目 [blog](https://github.com/zhuchuanlei/blog) 的源码

### 本地开发

1. 将`blog-code`源码克隆到本地
  ` git clone git@github.com:zhuchuanlei/blog-code.git `
2. 将`next`样式源码克隆到本地
  `git clone git@github.com:zhuchuanlei/hexo-theme-next.git themes/next`
3. 修改`next`配置
  * 启用`scheme: Mist`



切换至`develop`分支
